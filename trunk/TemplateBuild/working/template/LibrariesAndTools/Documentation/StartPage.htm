Alt.Net Web Application Template

This is the Application Solution for a common web application.  It uses a set of well known tools to create a useable and maintainable application stack.

 Getting Started:

  1. Build the Solution
  2. Run the ClickToBuild.bat file from the Root of the solution. This will create a new database and schema.

 

Documentation for the Application Stack

Frameworks:

  • UI framework: 
    • Asp.Net MVC http://asp.net/mvc
    •  MvcContrib http://MvcContrib.org
    • JQuery   http://jquery.com
  • Dependency Injection: StructureMap
  • Unit Test: NUnit http://nunit.org
  • UI Testing: Watin
  • Mocking Framework: Rhino Mocks
  • ORM: NHibernate

Tooling:

  • Build: NAnt  http://
  • Database Migrations: Tarantino
  • Continous Integration: CruiseControl.Net
